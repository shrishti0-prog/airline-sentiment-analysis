# Airline Tweet Sentiment Analysis ✈️

This project performs Sentiment Analysis on Airline Tweets using Natural Language Processing.

## Features

- Text preprocessing using spaCy
- TF-IDF Vectorization with n-grams
- Logistic Regression classifier
- Evaluation using F1-score
- Streamlit web app for real-time prediction

## Dataset

Airline Tweets dataset containing positive, negative and neutral tweets.

## Model

- TF-IDF (max_features=8000, ngram_range=(1,2))
- Logistic Regression (class_weight="balanced")
- Evaluated using weighted F1-score
